# Overview
 The data in this directory is generated by
 - Submitting data from test env forms
 - Open CDP terminal for test env
 - Logging into mongo shell and export data from relevant collection e.g. `print(EJSON.stringify(db.organisation.findOne({orgId:503152}, null, 2)))`
